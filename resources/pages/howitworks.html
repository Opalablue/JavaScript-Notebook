Where it appears on websites?

When writting code in JavaScript, it will appears in a webstite inside the <script> tag, that must be inside of the <body> tag.

Variables and types

In JavaScriptvariables are dynamic. That means you don't need to specify it's type, different of what happens with static languages like Java.

Yet, they can still have some of these primitive types:

  • number: 8.3 ; 10 ; 0.333
  • object: [] ; null
  • string: "Hello, World!" ; 'Programmed to WORK'
  • Boolean: true ; false
  • function: functionname(){}
  • null
  • undefined
