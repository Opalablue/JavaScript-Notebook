Where it appears on websites?

When writting code in JavaScript, it will appears in a webstite inside the <script> tag, that must be inside of the <body> or the <head> tag. For external JavaScript source, you should add the attribute src inside of the <script> tag. The file must have the extention .js.

Interpreted Language

Being an interpreted language means the code is executed from the top to bottom and the results are immediately returned. It doesn't need to be transformated before the browser runs it, unlike compiled languages like C and C++.

JavaScript also use a technique called just-in-time compilling to improve performance.

Dynamic and Static

Being a dynamic page means it will generate and display new content as required. Static pages are the ones that only display the same content all the time. JavaScript is commonly used to create dynamic pages.

Variables and types

In JavaScript variables are dynamic. That means you don't need to specify it's type, different of what happens with static languages like Java.

Yet, they can still have some of these primitive types:

  • number: 8.3 ; 10 ; 0.333
  • object: [] ; null
  • string: "Hello, World!" ; 'Programmed to WORK'
  • Boolean: true ; false
  • function: functionname(){}
  • null
  • undefined

OBS: JavaScript is Case Sensitive! It means there is a big diference between a variables called "myvariable" and "MyVariable"
