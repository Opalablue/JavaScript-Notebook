What is JavaScript?



JavaScript is a programming language commonly used to create the interactive part of websites, working on the client-side. It can also be used to make games or even running on the back-end of servers with the help of node.js.

JavaScript is Object Oriented, interpreted, multiplatform, object-based, supports first-class functions and is non-compiled.

Ecma International is the organization that controls the ECMAScript, the padronized version of JavaScript. Most browsers are compatible with ECMAScript 5.1, and the most modern ones are already compatible with ECMAScript 6.
