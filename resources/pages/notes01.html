Alerts, Confirmations and Questions

To emit an Alert window in JavaScript: 

                        
  1. <script>
  2.     window.alert("Message")
  3. </script>

To emit a Confirmation Signal in JavaScript: 

                        
  1. <script>
  2.     window.confirm("Are you liking JavaScript?")
  3. </script>
    4.

Output


To emit a Question signal in JavaScript: 

                        
  1. <script>
  2.     window.answer("What is your name?")
  3. </script>
    4.

Output


Comments

To show comments on your JavaScript code, you can use // for a single line comment or /* comment here */ to multi-line comments.
Comment lines are not executed and can be used to provide information about the piece of code.
